Sirius CP CLI

The ultimate unified local environment for competitive programming. Built in Python, designed to completely eliminate the friction of context-switching between your browser, your code editor, and the grading servers.

Installation

Ensure you have Python 3.9+ and a C++ Compiler (g++) configured natively or via WSL.

git clone https://github.com/amineaith3/sirius-cp-cli.git
cd sirius-cp-cli
pip install -e .

Because the suite uses a Monorepo architecture, this single installation gives you access to both the sirius-cf and sirius-cses commands globally.

Configuration

Configure the CLI to automatically organize your problems into specific offline directories.

Codeforces

sirius-cf init
  • Handle: Tracks your solved problems.
  • CF_COOKIE: Your Codeforces session string (Required for submitting).
  • Workspace Path: The absolute path where `sirius-cf` should download all contests. (Defaults to `~/Desktop/cf-offline`).

CSES

sirius-cses init
  • Handle: Your CSES username.
  • CSES_COOKIE: Your `PHPSESSID` cookie string (Required for submitting/syncing).
  • Workspace Path: The absolute path where `sirius-cses` builds your Categorized Dashboard. (Defaults to `~/Desktop/cses-offline`).

Codeforces Quickstart

1. Find your next problem

sirius-cf next 800

2. Fetch a specific problem

sirius-cf fetch 1800 B

3. Test your solution

sirius-cf test 1800 B

4. Submit and track

sirius-cf submit 1800 B

Codeforces Features

  • • Offline-First Fetching: Scrapes the problem statement, injects MathJax for perfect offline math rendering, and extracts sample test cases.
  • • Native & WSL Testing: Instantly compiles your C++ code natively (or via WSL if on Windows) and runs it against local test cases with line-by-line mismatch diffing.
  • • Live Direct Submission: Submits your solution directly to the Codeforces judging servers, and polls the live grading status in real-time.

CSES Quickstart

1. Build your local dashboard

Maps all 400 problems and updates your stats.txt tracking files.

sirius-cses sync

2. Fetch your next Graph problem

sirius-cses next graph

3. Test a solution

sirius-cses test 1068

4. Submit to CSES

If ACCEPTED, your stats.txt will automatically update!

sirius-cses submit 1068

CSES Features

  • • Categorized Workspace: Automatically creates human-readable directories like: cses-offline/Introductory Problems/1068--Weird-Algorithm/.
  • • Living Dashboard: A stats.txt file is dynamically generated in every category folder tracking your offline progress.
  • • Bulk Offline Mirror: Run download-all to safely mirror all 400 problems, test cases, and HTML statements locally to your machine.
