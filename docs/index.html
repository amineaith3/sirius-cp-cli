Sirius CP CLI

The ultimate unified local environment for competitive programming. Built in Python, designed to eliminate browser friction and fully automate offline fetch-test-submit workflows.

# Installation

Requires Python 3.9+ and g++ configured natively or via WSL.

git clone https://github.com/amineaith3/sirius-cp-cli.git cd sirius-cp-cli pip install -e .

Because the suite uses a Monorepo architecture, this single installation gives you access to both the sirius-cf and sirius-cses commands globally.

# Configuration

Codeforces

sirius-cf init
  • -- Handle: Tracks your solved problems.
  • -- CF_COOKIE: Your Codeforces session string.
  • -- Workspace Path: Absolute path for Codeforces data (e.g. ~/Desktop/cf-offline).

CSES

sirius-cses init
  • -- Handle: Your CSES username.
  • -- CSES_COOKIE: Your PHPSESSID string.
  • -- Workspace Path: Categorized Workspace location (e.g. ~/Desktop/cses-offline).

# Codeforces Quickstart

1. Find next problem

sirius-cf next 800

2. Fetch directly

sirius-cf fetch 1800 B

3. Test solution

sirius-cf test 1800 B

4. Submit & poll

sirius-cf submit 1800 B

# CSES Quickstart

1. Build Local Dashboard

Maps all 400 problems and generates `stats.txt` tracking.

sirius-cses sync

2. Fetch Next Problem

sirius-cses next graph

3. Submit

sirius-cses submit 1068
